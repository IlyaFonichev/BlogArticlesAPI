# Blog Articles API

FastAPI приложение для управления статьями блога.

## Функциональность

- Получение списка статей (с фильтрацией и пагинацией)
- Получение конкретной статьи по ID
- Создание новой статьи
- Обновление существующей статьи
- Удаление статьи
